Build Your Own Backend

TripPlanner is a database of users, countries and user trips. The links on this page are for all GET requets to the database.

Click on a link or copy and paste link at end of URL to navigate.

POST, PATCH and DELETE requests will require Postman or similar.

See the code on GitHub
